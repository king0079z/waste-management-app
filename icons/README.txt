PWA icons (required for "Add to Home Screen" / install prompt).

Add these files in this folder:
- icon-192.png   (192 x 192 pixels, PNG)
- icon-512.png   (512 x 512 pixels, PNG)

You can use your logo or any square image. Resize to 192 and 512 for best results.
Until these exist, install may still work on some browsers using the app icon from the tab.
